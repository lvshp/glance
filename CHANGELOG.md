- 新增 EPUB 阅读支持
- 新增 EPUB 章节目录、章节跳转与目录选择
- 新增阅读进度自动保存与自动恢复
- 新增多行显示与按页滚动
- 新增按终端宽度与中文显示宽度自适应折行
- 新增更偏 IDE / 工作台风格的伪装界面
- 新增多平台 Release 附件
- 新增自动发布 GitHub Release 的工作流

### 改进

- 优化 README 首页说明与下载入口
- 优化发布流程，支持校验文件输出
